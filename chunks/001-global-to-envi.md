# 001 全局机制到 ENVI/SET

文档范围：`PECMD帮助文档.txt` 第 1-932 行。

本块覆盖全局脚本规则、内置变量、WinCMD 控件概览，以及 `_END`、`_ENDFILE`、`_SUB`、`POS`、消息映射、代码段、`LAMBDA`、`ADSL`、`BASE`、`BROW`、`CALC`、`CALL`、`CHEK`、`CODE`、`COME/NOTE`、`CMPS`、`DATE`、`DEVI`、`DFMT`、`DIR`、`DISK`、`DISP`、`DTIM`、`EDIT`、`EJEC`、`ENVI/SET`。

## 全局脚本与命令机制

功能点：
- 支持命令行执行单条 PECMD 命令。
- 支持通过配置文件执行脚本，典型入口是 `LOAD` 或 `MAIN`。
- 支持多种脚本文本扩展名，包括 `WCE`、`WCI`、`WCS`、`WCX`、`INI`、`INF`、`TXT`、`LOG` 以及自定义格式。
- 支持压缩脚本或资源格式，如 `WCZ`、`WCM`、`WZM`。
- 支持脚本内整行注释和行后注释。
- 命令不区分大小写。
- 命令参数支持环境变量展开。
- 支持引号参数解析，用于包含空格或特殊字符的路径/参数。
- 支持命令行模式和脚本模式不同的注释默认开关。
- 支持 `NOTE` / `COME` 切换注释解析。
- 支持命令主体字符集：字母、数字、下划线、汉字。
- 支持命令前若干个 `^` 表示预先解释本命令多次。
- 支持 `%%` 表示字面 `%`，避免变量展开。
- 支持一行中调试专用 `-[ ]LOGS` 信息，只在日志中输出。
- PECMD 退出码为最后命令错误码；负数或大于 `0x7FFFFFFF` 映射为进程退出码 `1`。

隐藏功能：
- `EXEC*1 ... NAME=命令` 可在收到一行信息后立即终止执行。
- 建议优先使用 PE 变量而非环境变量，以便多线程环境中隔离状态。
- 在 CMD 批处理中引用 PECMD 内置变量需要双百分号。
- 命令参数中含 `>`、`|`、`&`、`^` 时需要引号或转义。
- 盘符整理类命令必须放在 `PECMD.INI` 最前面。

Lua 重构提示：
- Lua 入口应明确区分“兼容脚本执行器”和“现代 Lua API”。
- 兼容脚本执行器需要保留命令名不区分大小写、变量展开、注释规则、引号规则和错误码语义，除非明确废弃。
- 现代 Lua API 不应复刻脆弱语法，但需要提供同等能力。

遗漏风险：
- 全局规则不出现在单个命令块中，但会影响所有命令兼容性。

## WinCMD 控件总览

功能点：
- 支持创建窗口。
- 支持 ComboBox 下拉选择框。
- 支持 Button 按钮。
- 支持 Picture/IMAG 图像区域。
- 支持 CheckBox 复选框。
- 支持 Edit 编辑框。
- 支持 HotKey 系统热键。
- 支持 Popup-Menu 托盘图标菜单。
- 支持 Radio 单选框。
- 支持 ProgressBar/PBAR 进度条。
- 支持 Group 分组面板。
- 支持 Static/LABE 文本标签。
- 支持 Timer 定时器。
- 支持 Tabl 表格。
- 支持 List 列表/下拉列表。
- 支持 MEMO 文本编辑控件。
- 支持 BROW 浏览窗口。

Lua 重构提示：
- GUI 能力可现代化为 ImGui 或其他 UI 层，不必复刻 Win32 控件语法。
- 但需要确认旧脚本依赖的控件状态读写、事件回调、消息映射是否有替代能力。

## 内置变量

功能点：
- 提供当前配置文件目录、文件名和驱动器变量：`CurDir`、`CurFile`、`CurDrv`。
- 提供用户 Shell 目录变量：`Favorites`、`IECache`、`Personal`、`Programs`、`SendTo`、`StartMenu`、`Startup`、`QuickLaunch`、`Desktop`。
- 提供自身程序名：`MyName`。
- 提供主脚本标志：`&&__MAIN__`。
- 提供启动或 LOAD 前目录：`&__OldDir`。
- 提供命令错误码：`&&ERROR`、`ERROR`。
- 提供 PE 消息与控件 ID 起始值：`&WM_PE_BASE`、`&PE_IDBASE`、`&PE_MENU_IDBASE`。
- 提供进程、父进程、线程、最近创建线程/进程、实例句柄：`&__PID`、`&__PPID`、`&__TID`、`&__LastPID`、`&__LastTID`、`&__HINST`。
- 提供 `EXEC` 等待程序退出码：`&&ERRORLEVEL`。
- 提供 32/64 位状态：`&bX64`。
- 提供版本信息：`&PECMDVER`、`&PECMDBUILD`。
- 提供随机数：`RANDOM`。
- 提供实时当前目录：`_CD`、`&_CD`。
- 提供当前窗口和最近窗口 ID：`&__WinID`、`&__LastWinID`。
- 提供 `WM_NOTIFY` 参数变量：`&__NMHDR.idFrom`、`&__NMHDR.code`、`&__NMHDR.hwndFrom`。
- 提供当前日志文件：`&__LOGS`。
- 提供 `WM_COMMAND` 参数变量：`&__wParam.wID`、`&__wParam.wNotifyCode`。
- 提供当前 PE 栈唯一 cookie：`&__THIS`。
- 提供函数返回约定变量：`&&__RET`。
- 支持 `%PeVar?类型:[~]off%` 读取 PE 变量二进制对象中的指定类型数据。

Lua 重构提示：
- 这些变量应拆成 `context`、`env`、`process`、`ui`、`runtime` 几类上下文对象。
- `RANDOM`、错误码、进程 ID、当前脚本路径等应作为 Lua API 暴露。

## _END

功能点：
- 结束 `_SUB` 定义的子过程。
- 调用子过程结束后继续执行调用方 `CALL` 的下一条命令。
- 参与脚本级子过程边界管理。

隐藏功能：
- `_SUB` 与 `_END` 之间的命令只由对应 `CALL` 执行，主流程跳过。
- `_SUB` 和 `_END` 必须一一匹配。
- 子过程不能嵌套定义子过程。
- 子过程可放在配置文件任意位置，被同文件 `CALL` 调用。
- 只能在配置文件中使用，不能命令行使用。

Lua 重构提示：
- 由 Lua `function ... end` 替代。
- 兼容脚本解析器如保留旧脚本支持，需要实现主流程跳过子过程体。

## _ENDFILE

功能点：
- 标记脚本逻辑结尾。
- 结尾之后的内容不再调入或执行。
- `_ENDFILE-IMPORT` 只对 `IMPORT` 的文件有效，可使后续内容被丢弃。

Lua 重构提示：
- 现代 Lua 可用 `return` 或文件边界替代。
- 兼容加载器需要识别此标记，避免把后续附加数据当脚本。

## _SUB / 类 / 窗口 / 函数

功能点：
- 定义子过程/函数。
- 定义类或嵌套子类。
- 定义窗口。
- 支持 `this` 函数调用语义。
- 支持持久栈函数。
- 支持隐藏窗口。
- 支持函数析构命令，在返回前自动执行。
- 支持窗口位置、大小、标题、关闭事件、图标、透明度、无标题栏、无边框、透明色、异形蒙板等窗口属性。
- 支持窗口状态控制：置顶、强制不最小化、禁用最小化、启用最大化、禁用关闭按钮、无系统菜单、无标题栏、无焦点、禁用 Alt 拖动、可调整大小、关闭按钮不退出、无 Tab 焦点、客户区尺寸、非激活、DPI 缩放、XP 缩放、禁用 XP 视觉。
- 支持窗口蒙板和背景图。
- 支持窗口和控件名字作为可操作对象。
- 支持类实例，一个类可以生成多个实例，每个实例有自己的名字。
- 支持父类成员函数访问和多级类域 `::` 访问。
- 支持普通函数作为窗口对象创建多个对象/纤程。

隐藏功能：
- 名称第一个字符不能是 `$`。
- 名称第一个字符为 `&` 表示局部名称，线程中不冲突。
- 多个 `_SUB` 定义的名称不能重复、不能近似，也不能用这些名称设置成环境变量。
- `_SUB` 扩展后可添加自定义命令，不需要 `CALL` 即可像内部命令一样调用。
- 退出时自动销毁 PE 变量、锁、控件、HKEY。
- 约定 `.` 为类数据成员标志。
- 设置窗口透明度后，`IMAG` 控件无法显示 GIF 动画。
- 可通过 `ENVI @窗口名称=标题` 修改窗口标题。
- 可通过 `ENVI @窗口或控件.Enable` 设置启用状态。
- 可通过 `ENVI @窗口或控件.Visible` 设置显示状态；`*` 为第 2 种方案。
- `Visible >= 1` 可映射 `SW_SHOW`、`SW_SHOWNORMAL`、`SW_MAXIMIZE`、`SW_MINIMIZE`、`SW_RESTORE`、`SW_SHOWDEFAULT`、`SW_SHOWMAXIMIZED`、`SW_SHOWMINIMIZED`、`SW_SHOWMINNOACTIVE`、`SW_SHOWNA`、`SW_SHOWNOACTIVATE`、`SW_FORCEMINIMIZE`。
- 没有标题的窗口可以按住 Alt 和鼠标左键拖动。
- 可通过 `ENVI @窗口.Paint=函数名` 设置画布回调。
- 可通过 `ENVI @窗口或控件.InvalidateRect` 刷新区域。
- 可通过 `ENVI @窗口或控件.Cursor` 设置光标。
- 可通过 `ENVI @窗口或控件.Font` 设置字体。
- 可通过 `ENVI @窗口或控件.bkcolor` 设置背景色。
- 可通过 `ENVI @窗口或控件.trans` 设置透明模式。
- 可通过 `ENVI @窗口或控件.style` 修改窗口风格。
- 可通过 `ENVI @窗口.HitTest` 设置可拖动区域。
- 可通过 `ENVI @窗口或控件.cmd` 动态设置响应命令；查询模式需要 `ENVI^ QueryCmd=1`。
- 可通过 `ENVI @@Enable`、`@@Visible`、`@@IsWindow` 用窗口 ID 操作或查询窗口状态。

Lua 重构提示：
- 需要拆成 Lua 函数、对象生命周期、UI 窗口/控件、事件系统、资源自动回收几个能力。
- 不建议复刻 `_SUB` 类系统语法，但 Lua API 必须覆盖函数、对象实例、析构、事件绑定、UI 状态控制。

## POS / 位置 / 大小 / 层级

功能点：
- 通过 `ENVI @窗口或控件.POS` 设置窗口/控件位置、大小、层级、透明度、前置、激活、父窗口、客户区大小。
- 通过 `ENVI @窗口或控件.POS=?...` 查询窗口/控件位置、大小、客户区坐标、激活状态、父窗口 ID。
- 支持通过窗口 ID 使用 `ENVI @@POS` 设置或查询。
- 支持屏幕绝对坐标查询。
- 支持实际宽高查询。
- 支持层级语义：底层、取消最顶层、顶层、钉在最顶层。

Lua 重构提示：
- 映射为 `ui.window.set_bounds`、`ui.window.get_bounds`、`ui.window.set_z_order`、`ui.control.set_bounds` 等。

## 消息映射

功能点：
- 为窗口或控件定义 Windows 消息映射。
- 支持前部命令、后部命令、替代命令、不执行系统响应、返回结果码、超级捕捉等模式。
- 支持异步发送消息 `POSTMSG`。
- 支持同步发送消息 `SENDMSG`。
- 支持通过窗口名/控件名或窗口 ID 发送消息。
- 支持消息号为 PECMD 应用自定义消息。
- 支持命令接收 `wParam`、`lParam` 参数名。
- 支持 `COMMAND` / `NOTIFY` 控件消息格式。
- 支持消息参数为数字、PE 变量、字符串。

隐藏功能：
- `EXEC` 可将命令输出实时显示到控件，如 `EDIT`、`MEMO`、`LABE`。
- 窗口号可用通配查询，带等待超时和间隔。

Lua 重构提示：
- 需要事件总线和 Win32 消息桥接。
- ImGui 方案中应提供抽象事件，不必完全暴露 Win32 消息，但兼容层可能需要。

## 代码段 `{}`

功能点：
- 用 `{}` 定义一段多行代码。
- 代码段有自己的堆栈。
- 退出时自动销毁 PE 变量、锁、控件、HKEY。
- 可作为内部函数或命令组使用。
- `{*` 表示 this 代码段，没有自己的堆栈和自动销毁功能，只作范围指示。
- 支持文件级、函数内、命令组内代码段。
- 支持嵌套。
- 支持条件代码段，与 `IFEX`、`FIND` 配合提供 if/else 多行结构。
- 支持 `LOOP`、`FORX` 的多行代码体。

Lua 重构提示：
- 由 Lua 块、函数、闭包和作用域自然替代。
- 若要执行旧脚本，解析器需要保留代码块和自动资源释放语义。

## LAMBDA

功能点：
- 定义带参数表的匿名/内联代码函数。
- 有自己的堆栈。
- 退出时自动销毁 PE 变量和锁。
- 可在文件级、函数内、命令组中使用。
- 支持嵌套。
- 命令组级 LAMBDA 函数体作为变量串解释。
- `_SUB` 函数本质也是 LAMBDA。
- LAMBDA 可访问调用者堆栈。

Lua 重构提示：
- 直接映射 Lua 匿名函数和闭包。

## ADSL

功能点：
- 建立 ADSL 宽带连接。
- 使用 BASE 生成的变异 Base64 编码账号和密码拨号。
- 支持拨号重试次数。
- 支持指定或自动选择宽带连接名称。
- 系统中不存在有效宽带连接时可创建默认或指定连接。
- 账号或密码省略时只建立连接不拨号。
- 支持停止拨号连接。
- 支持列出连接。
- 支持 WLAN 配置：SSID、密码、加密方式、序号。
- 支持 WLAN list/query/scan，返回 SSID、信号质量、安全、连接状态、PHY 类型等信息。

Lua 重构提示：
- PPPoE/RAS 与 WLAN 应拆成 `net.ras` 和 `net.wlan`。
- 现代 WinPE 是否保留 ADSL 需要评估；WLAN 查询/连接可能仍有价值。

## BASE

功能点：
- 计算 PECMD 变异 Base64 编码，主要用于 ADSL 账号密码。
- 可将结果保存到变量。
- 省略变量时显示结果并复制到剪贴板。
- 输入字符串限制 256 字符且不能含逗号。
- `BASE*` 表示常规 Base64 算法。
- `BASE* -u` 提供常规 Base64 解码。

Lua 重构提示：
- 提供 `codec.base64_encode/decode`。
- PECMD 变异算法如需兼容 ADSL 旧脚本，应单独实现为 `pecmd_base64_variant`。

## BROW

功能点：
- 打开文件选择对话框。
- 打开保存文件对话框。
- 打开目录选择对话框。
- 保存选择结果到变量。
- 支持初始路径、提示文字、扩展名过滤、多过滤器串。
- 支持目录对话框附加标志：编辑框、新建文件夹按钮、混合选择文件和目录。
- 支持文件对话框标志：多选、浏览器风格、新建警告、排除最近访问快捷方式、可变大小、文件必须存在、短文件名、覆盖警告、只读勾选、无效路径警告、排除只读文件、去掉只读勾选器、选择只读或隐藏文件。
- `-fix` 尝试解除系统目录屏蔽。
- 不再影响当前工作目录。

Lua 重构提示：
- 映射为 `ui.dialog.open_file`、`save_file`、`select_folder`。

## CALC

功能点：
- 计算表达式并保存到变量或显示结果。
- 支持整数和浮点模式。
- 支持输出进制：2、8、10、16、任意 N 进制。
- 支持 GUI 计算器模式。
- 支持错误值配置。
- 支持多个表达式，用分号或换行分隔。
- 支持科学计数法输入。
- 支持单位后缀 `T/G/M/K/S`。
- 支持逻辑、比较、位运算、加减乘除、取余、幂、括号优先级。
- 支持数学常量 `e`、`pi`。
- 支持格式化输出：科学计数法、小数、自动简洁格式、小数位数、保留 0、四舍五入。
- 支持函数：绝对值、三角/反三角、取整、对数、最大最小、幂、随机、位移、逻辑反、按位反等。
- 支持向量操作。
- 支持通过 `%RANDOM%`、`RAND`、`ROUND()` 获取随机数。

Lua 重构提示：
- Lua 已有数值表达能力，但如果要兼容旧表达式语法，需要独立表达式解析器。
- 现代 API 可直接使用 Lua 表达式或提供安全表达式求值器。

## CALL

功能点：
- 调用 DLL 函数。
- 调用 `_SUB` 子过程。
- 调用窗口或窗口类实例。
- 支持 DLL 预加载、释放、查询函数地址、地址调用。
- 支持 `LoadLibrary`、内存 DLL、`GetProcAddress`、`FreeLibrary`。
- 支持 `rundll32` 风格调用。
- 支持 CPL 控制面板文件调用。
- 支持 Pascal/stdcall 默认调用和 C 调用规则。
- 支持最多 20 个普通函数参数，C 调用可支持更多。
- 支持 Unicode 字符串、窄字符串、整数、PE 变量、原始串、UTF-8 串等参数类型。
- 支持返回值保存、布尔返回、符号数返回、十六进制返回。
- 支持执行前切换到目标 DLL 目录，执行后恢复或不恢复目录。
- 支持注册/反注册 DLL。
- 支持调用子过程时传递参数 `%0`、`%1`、`%#`、`%*`、`%~n`、`%@`。
- 支持省略 `CALL` 直接调用自定义命令。
- 支持窗口调用模式：同步、并行、背景、后台、弃儿窗口、父窗口不遮挡子窗口。
- 支持生成窗口类实例。
- 支持外置子窗口。
- 支持弹出式菜单调用。
- 支持内存动态函数代码执行。
- 支持初始化或卸载 Win 环境。

Lua 重构提示：
- DLL 调用应由 LuaJIT FFI 或 libffi 封装。
- 子过程/窗口调用应由 Lua 函数和 UI 对象系统替代。
- 弹出菜单、窗口实例、后台调用需要事件调度/协程模型。

## CHEK

功能点：
- 创建复选框控件。
- 支持位置大小、标题、点击事件、初始勾选状态、灰色禁用、不可见。
- 支持右对齐、居中、缩放和图片相关选项。
- 支持退出代码块或函数时自动回收。
- 可通过变量读取复选框标题。
- 可通过 `ENVI @复选框.Check` 设置勾选状态。
- 可通过 `ENVI @复选框.Enable` 设置启用状态。
- 可通过 `ENVI @复选框.Visible` 设置可见状态。
- 可通过 `ENVI @控件.*del` 删除控件。

Lua 重构提示：
- 映射为 UI checkbox 控件和状态绑定。

## CODE

功能点：
- 文件或变量内容编码转换。
- 支持不输出 BOM。
- 支持只翻译 GBK/BIG5 字符但不改编码。

## ENVI / SET

功能点：
- 设置、清除环境变量。
- 设置、清空、创建、复制、比较、交换 PE 变量。
- `SET` 是 PE 变量版，等价于 `ENVI &` 或本级 `ENVI &&`。
- 支持进程环境变量、系统环境变量、用户环境变量。
- 支持广播环境变量变更。
- 支持后台广播或只设置不广播。
- 支持全局 PE 变量。
- 支持函数/窗口局部 PE 变量。
- 支持本级变量遮盖上级变量。
- 支持变量析构函数，退出作用域时调用。
- 支持二进制 PE 对象：创建、dummy 创建、固定 dummy、清零、按类型写入、按类型读取。
- 支持 Unicode 与多字节互转。
- 支持返回内存地址和长度。
- 支持二进制对象部分拷贝和比较。
- 支持值作为十六进制或十进制字符序列转换为字符串/字节。
- 支持变量再解释一次或多次。
- 支持向量操作。
- 支持环境变量存在性检查。
- 支持强制局部变量模式 `FORCELOCAL`。
- 支持 PE 变量继承开关 `EXPORTLOCAL`。
- 支持变量解释模式 `EnviMode`。
- `EnviMode=0` 为兼容模式：空变量不解释，并自动多轮次非顺序解释。
- `EnviMode=1` 为标准模式：未定义变量解释为空，顺序解释一遍，但 `=` 前变量名仍可多轮解释。
- 支持刷新环境变量。
- 支持自动写入 CDROM 盘符环境变量。
- 支持设置并广播 Shell 目录环境变量：Favorites、Desktop、StartMenu、Startup、Programs、SendTo、Personal、QuickLaunch。

隐藏功能：
- 可用 `ENVI @@DeskTopFresh` 刷新桌面、我的电脑、图标缓存、文件关联、目录。
- 可用 `ENVI @@TaskIcoMenu` 开启、关闭、切换托盘菜单。
- 可用 `ENVI^ HelpColor` 设置 HELP 默认颜色。
- 可用 `ENVI^ Arg` 重写/分断函数参数表。
- 可用 `ENVI^ Clipboard` 设置剪贴板。
- 可用 `ENVI^ Clipboard?` 查询剪贴板。
- 支持前后空字符，但命令尾部空字符可能被丢弃；可用 `TEAM` 或变量保留尾部空白。
- 支持 `~`、`~~`、`~.`、`~*` 递归解释变量名或函数参数，并可反复调用。
- PE 变量退出定义范围时自动销毁，但只能清空和修改，不能删除，以保证共享引用稳定。
- 全局 PE 变量所有函数和线程都可访问，程序运行期间始终有效。
- 线程创建后会自动复制共享点之后的 PE 变量。
- 窗口和有持久栈函数的变量共享，普通函数或代码块变量非共享，子线程自动复制一份。
- 支持值传递和引用传递；引用传递可支持二进制数据和二进制 0，值传递不能表达二进制 0。
- 可用 `ENVI ?...=FVER,文件名` 查询文件版本和产品版本。
- 可用 `ENVI ?...=DROPFILE,wParam` 取得拖入文件名。
- 可用 `ENVI @@Cur` 移动或查询鼠标位置、显示隐藏鼠标。
- 可用 `ENVI @窗口.Cur` 移动或查询窗口客户区鼠标位置。
- 可用 `ENVI @窗口或控件.ID` 查询窗口 ID 和控件 ID。
- 可用 `ENVI ?...=FPOS,文件名` 查询文件磁盘位置、簇大小、磁盘块分布、块数、分区起始、设备返回名。
- 可用 `ENVI ?...=PWIN,x,y,标志` 查询某点下的窗口。
- 可用 `ENVI @窗口或控件.MouseCapture` 开启或释放鼠标捕获。
- 可用 `ENVI @@EATEKEYS` 拦截组合键。
- 可用 `ENVI @@RMENU` 获取文件右键菜单。
- 可用 `ENVI^ Alias` 设置命令别名。
- 可用 `ENVI^ EnviBroad` 开关环境变量广播。
- 可用 `ENVI^ __arg` 开关兼容精简版参数表。

Lua 重构提示：
- 这是旧 PECMD 的最大“隐藏功能集合”，不能只当作环境变量命令。
- 现代化应拆成 `env`、`pevar/buffer`、`clipboard`、`desktop`、`mouse`、`window`、`file_info`、`alias`、`arg_parser` 等模块。
- 若不实现 PE 变量二进制对象，需要明确提供 LuaJIT FFI buffer 替代方案。
