# 004 RPOS 到 WRIT

文档范围：`PECMD帮助文档.txt` 第 2639-3331 行。

本块覆盖 `RPOS`、`RSTR`、`RUNS`、`SCRN`、`SED`、`SEND`、`SERV`、`SHEL`、`SHOW`、`SHUT`、`SITE`、`SIZE`、`SLID`、`SOCK`、`SPIN`、`SSTR`、`STRL`、`SUBJ`、`SWIN`、`TABL`、`TABS`、`TEAM`、`TEMP`、`TEXT`、`THREAD/THRD`、`TIME`、`TIPS`、`UPNP`、`USER`、`WALL`、`WAIT`、`WRIT`、并行符、DLL 版 PECMD。

## RPOS

功能点：
- 从右侧查找字符或子串出现位置。
- 支持查找第 N 次出现。
- 支持返回多个位置。
- 支持从右计算位置。
- 支持大小写匹配。
- 支持被查找字符或字符串通过变量传入。
- 支持子串模式。
- 按 Unicode 字符串处理。
- 找不到返回 0。

Lua 重构提示：
- 映射为 Unicode-aware `string.rfind`。

## RSTR

功能点：
- 从右侧截取指定数量字符。
- 数量小于 1 或超出源字符串长度时返回整个字符串。
- 源字符串可直接给出或通过变量传入。
- 按 Unicode 字符串处理。

Lua 重构提示：
- 映射为 Unicode-aware right substring。

## RUNS

功能点：
- 设置 Windows 启动项。
- 程序命令可为 EXE、CMD、BAT 等可执行命令，并可带参数。
- 启动项名称为文本。
- 程序命令和启动项名称中的逗号可用全角中文逗号或变量传入规避。
- 只能在配置文件中使用，不能在命令行中使用。
- 本质可由 `REGI` 写 Run 项实现。

Lua 重构提示：
- 映射为 `startup.add_run_entry` 或注册表封装。

## SCRN

功能点：
- 返回当前屏幕宽高。
- `-win` 返回最大窗口大小。
- 可返回任务栏位置。
- 支持屏幕捕捉。
- 屏幕捕捉支持 GUI 模式。
- 屏幕捕捉支持指定格式，如 `image/png`。
- 支持捕捉整个屏幕、指定窗口 ID、指定矩形区域。
- 可带源文件。
- 无位置参数和窗口 ID 时获取先前保存内容。
- 无文件名时留到下次再取。

Lua 重构提示：
- 映射为 `display.metrics`、`screen.capture`。

## SED

功能点：
- 正则替换。
- 正则查找位置。
- 支持大小写不敏感。
- 支持返回多个位置。
- 支持替换次数和跳过次数。
- 支持全部替换。
- 支持将匹配结果写入多个子名。
- 支持从右计算位置。
- 支持字符集翻译。
- 支持字符串集翻译。
- 支持原串、正则串、替换串来自变量。
- 支持分行操作。
- 支持向量操作。
- 支持非开头时的引导串和非结尾时的结尾串。
- 支持把 `\u` 转换为大写、`\l` 转换为小写。
- 标志串三字符分别控制后面三个串是否解释或按字面。

Lua 重构提示：
- 映射为 `regex.replace/find`、`text.translate`。
- Lua 原生 pattern 与正则不等价，兼容层需选择 PCRE/Oniguruma/Windows 正则等实现。

## SEND

功能点：
- 模拟键盘按键。
- 模拟鼠标事件。
- 支持 GUI 模式。
- 支持不抢焦点、指定右/左/顶层行为。
- 支持扩展键。
- 支持安静模式。
- 支持按下、弹起、按下并弹起。
- 支持简单字母数字按键。
- 支持 `#虚拟键码`、十六进制、十进制、`VK_` 名称。
- 支持模拟 Alt+Tab 等组合键。
- 鼠标事件支持绝对坐标和相对坐标。
- 鼠标事件支持移动、左/右/中键按下与抬起、X 键、滚轮、映射整个虚拟桌面、触摸屏。
- 鼠标事件支持 dx/dy、滚轮数据、扩展数据。

Lua 重构提示：
- 映射为 `input.send_key`、`input.send_mouse`。

## SERV

功能点：
- 启动服务或驱动程序。
- 停止服务或驱动程序。
- 查询服务或驱动程序。
- 创建服务。
- 删除服务。
- 删除服务时可自动停止，也可禁用自动停止。
- 支持等待模式。
- 支持设置启动类型：boot、system、auto、demand、disabled、delayed-auto。
- 支持返回错误码。
- 创建服务支持服务名称、文件、类型、启动、出错处理、依赖、用户、密码、显示名、Group、Tag。
- 查询返回服务类型、当前状态、可接受控制、Win32 退出码、服务特定退出码、检查点、等待提示、进程 ID、服务标志。

Lua 重构提示：
- 现有 `win-utils.sys.service` 是候选实现。

## SHEL

功能点：
- 加载指定 Shell 并锁定。
- Shell 被杀时自动重新加载。
- 支持用户级 Shell，与 `MAIN -user` 配合。
- 支持强制为系统 Shell。
- 支持执行文件路径或以 `TEAM`/`EXEC` 开头的命令。
- 支持密码 BASE 字符串和重试次数。
- 省略密码时自动登录，默认重试 3 次。
- 可通过 `-shel:` 指定自动命令。
- 类似 `EXEC $`，会修改进程中的关机代码并 hook `ExitWindowsEx`。
- 只能在配置文件中使用。
- 要求在 `HOTK` 和 `HIDE` 后使用。

Lua 重构提示：
- 映射为 shell host/guardian；ExitWindowsEx hook 建议废弃或隔离。

## SHOW

功能点：
- 显示系统存在的移动硬盘、固定硬盘隐藏分区或未加载分区。
- 给这些分区分配盘符。
- 支持卸载盘符。
- 支持局部模式，其他进程可能不可见。
- 支持持久加载。
- 支持卸载全部或最近加载。
- 支持“如果已经加载则不加载”。
- 支持检查有效系统后再加载。
- 支持跳过指定类型、分区类型、物理分区。
- 支持从指定盘符表开始。
- 支持固定硬盘、U 盘/USB 移动硬盘标识。
- 支持按逻辑分区号或物理分区号选择。
- 分区号小于 0 表示全部无盘符分区。
- 分区号 0 表示全部无盘符可见分区。
- 省略盘符或盘符不合理时自动分配。
- 支持汉字作为“盘符”，用于解决盘符不足和隐藏加载。
- 支持设备就绪等待时间。
- 省略所有参数不会主动载入隐藏分区，载入隐藏分区时分区号必须为 -1。

Lua 重构提示：
- 映射为 `disk.mount.assign/remove/list`、`automount`。
- 现有 `win-kit.automount` 覆盖部分自动挂载逻辑。

## SHUT

功能点：
- 关闭系统。
- 重启系统。
- 注销用户。
- 休眠。
- 挂起。
- 锁定计算机。
- 弹出光驱并等待。
- 关闭光驱。
- 支持快速/强制关机。
- 支持另类 `SHUTDOWN` 模式，参数含关机、重启、强制、取消强制、延时秒数。
- 支持 `EXEC` hook `ExitWindowsEx` 后拦截开始菜单等关机调用。
- 可通过注册表 `FORCESHUTDOWN` 让开始菜单快速关机。
- 关机时自动执行 `%SystemRoot%\System32\OnShutdown.wcs`，并传入操作码和脚本参数。
- 操作码包括 shutdown、reboot、logout、suspend、hiber、poweroff、unknown、lock。

Lua 重构提示：
- 现有 `win-kit.poweroff` 覆盖部分关机前处理。
- 关机 hook 和脚本回调需明确是否保留。

## SITE

功能点：
- 设置或清除文件/目录属性。
- 支持 A、H、R、S：归档、隐藏、只读、系统。
- `+` 设置属性，`-` 清除属性。
- 支持更新文件时间。
- 支持创建时间、写入时间、访问时间。
- 支持本地时间、系统时间、UTC 时间模式。
- 支持查询文件版本和产品版本。
- 支持查询文件时间。
- 支持返回 UTC 时间整数，便于直接比较。
- 支持返回固定宽度字符串时间：年月日时分秒微秒星期。
- 支持周数扩展。
- 支持查询文件属性 A/H/R/S 和完整属性十六进制。
- 失败返回空。
- `FVER`、`FTIME`、`FATTR` 映射到 `ENVI/SET`。

Lua 重构提示：
- 映射为 `fs.attributes`、`fs.touch`、`version.query`。

## SOCK

功能点：
- 定义自动回收网络 socket 对象。
- 支持 TCP socket，默认 AF_INET/SOCK_STREAM/IPPROTO_TCP。
- 支持 UDP socket。
- 支持逗号或分号分隔参数。
- 支持检测对象存在。
- 支持禁止重复创建。
- 支持打开自动释放文件句柄供 API 使用。
- 文件句柄支持可写、读写删除共享、重置共享。
- 支持申请共享内存，可写、源文件、偏移、初值、初填长度。
- 支持申请事件量。
- 支持申请信号量。
- 支持申请互斥锁。
- 支持邮槽，`.read/.write` 收发。
- 支持命名管道，`.read/.write` 收发，`.connect` 连接。
- 管道连接模式支持立即连接、客户端、服务端、自动。
- 支持 `gethostbyname` 返回 IP 地址、多个地址或整数地址。
- 通过 `ENVI @名称.connect` 连接服务器并返回错误码。
- 通过 `ENVI @名称.write` 写数据，支持长度变量、数据变量、发送字节数、偏移、标志、IP、端口。
- 通过 `ENVI @名称.read` 读数据，支持长度变量、数据变量、接收字节数、偏移、标志、对方 IP/端口返回。
- read 支持自动多次读取超长数据。
- IP 名前加 `#` 可返回地址对象和对象长度。
- 通过 `ENVI @名称.close` 主动释放连接。
- 通过 `ENVI @名称.sock` 重新生成 socket。
- 通过 `ENVI @名称.shutdown` 友好关闭接收、发送或双向，且 `.sock/.close` 含该功能。
- 通过 `ENVI @名称.bind` 绑定监听 IP 和端口。
- 通过 `ENVI @名称.listen` 开始监听，默认连接池大小 7。
- 通过 `ENVI @名称.accept` 接受连接并返回对方 IP 和端口。
- 通过 `ENVI @名称.fd` 返回内部连接号，供 accept/select/Windows API 使用。
- 通过 `ENVI @名称.mem` 返回共享内存地址，可用于 `ENVI-mkfixdummy`。
- 支持 `setsockopt`。
- 支持 `ioctlsocket`。
- 支持 `DeviceIoControl`。
- 支持 `select`，可分别等待读、写、异常 fd，超时微秒，`.` 表示自身。
- 支持 `getname` 返回本方或对方 IP/端口。
- 支持等待信号量、事件、互斥锁，可通过对象或 `@@wait` 按类型和共享名等待。
- 支持设置事件/信号量，可通过对象或 `@@setevent` 操作 event/mutex/semaphore。

Lua 重构提示：
- 这是对象式兼容层，建议拆成 `net.socket`、`ipc.shared_memory`、`ipc.event`、`ipc.semaphore`、`ipc.mutex`、`ipc.pipe`、`ipc.mailslot`、`handle.file`。

## STRL

功能点：
- 返回字符串长度。
- 默认返回 Unicode 字符数，汉字算 1。
- `*` 表示后面是变量名，变量不存在时返回 -1。
- `-m` 返回多字节码字节数，汉字一般算 2。
- `-8` 返回 UTF-8 字节数。
- `-bin` 返回 PE 变量对象的内存字节数。

Lua 重构提示：
- 映射为 `utf8.len`、`encoding.byte_len`、`buffer.size`。

## SUBJ

功能点：
- 将路径与驱动器号关联，相当于 `SUBST`。
- 删除虚拟驱动器。
- 查询虚拟驱动器映射。
- 支持通知盘符增加或删除。
- 支持局部模式，其他进程可能不可见。
- 支持持久加载。
- 支持卸载全部或最近加载。
- 支持按目标路径卸载。
- 虚拟驱动器可为中文。
- 路径可为物理设备、CDROM、TYPE 设备、Volume GUID 路径。
- 一个盘符可连续加载不同位置，看到最后一个。
- 一个设备可多次加载。
- 文档警告删除盘符必须准确，否则可能删除物理驱动器映射。

Lua 重构提示：
- 映射为 `disk.define_dos_device`、`disk.mount_stack`。

Lua 重构提示：
- 映射为 UI child window / embedded panel。

## TEAM

功能点：
- 按顺序执行命令组。
- 多条命令默认用 `|` 分隔。
- 支持替代定界符：`^ # ~ + - @ $ % * ? / \ ; , . " ' <` 等。
- `<` 表示反向执行。
- 支持嵌套，最外层 `|`，内层 `||`，再内层 `|||` 依次增加。
- 不能与 `IFEX` 或 `FIND` 嵌套使用。

Lua 重构提示：
- 由 Lua 语句块或兼容脚本解析器替代。

## TEMP

功能点：
- 清理用户临时文件夹。
- 重新设置用户临时文件夹位置。
- 可静默清理。
- 可同时设置系统临时变量。
- 可生成唯一临时目录。
- 可生成唯一临时文件。
- 支持退出时自动删除生成的临时目录/文件。
- 支持模板前缀和尾部，尾部不能含目录。
- 可返回旧的或新设的临时目录。
- 从注册表读取临时目录位置。
- 文档建议不要在配置文件使用，必须启动到桌面使用。

Lua 重构提示：
- 映射为 `temp.cleanup`、`temp.set`、`temp.mktemp`、`temp.mkdtemp`。

## TEXT

功能点：
- 在登录画面或桌面窗口显示文字。
- 支持多行文字，`\n` 换行。
- 后缀 `.` 为原始模式，不转换换行。
- 支持颜色、左上右下矩形、右对齐、水平居中、垂直居中、字体大小、字体名。
- 文字为空时清除最近定义矩形区文字。
- 结尾 `*` 表示显示新文字前不清除原文字。
- 登录期间在登录画面显示，登录后在桌面窗口显示。
- 支持参数通过变量传入。

Lua 重构提示：
- 映射为 splash/overlay text。

## THREAD / THRD

功能点：
- 启动线程运行一条命令或命令组。
- 支持强制命令变为 PE 变量，便于多线程。
- 支持保持父子关系，包括窗口。
- 默认独立模式，彼此无关系。
- 支持弃儿线程，程序结尾不等待其退出。
- 支持提前解释命令组。
- 支持等待线程结束。
- 支持等待时不阻塞消息。
- 支持在当前栈作为子线程运行，从而可修改执行栈临时 PE 变量。
- 支持指定线程栈大小。
- 支持返回线程 ID 和线程句柄变量。
- 支持进程结束前等待该线程。
- 创建线程后 `%&&__LastTID%` 为刚创建线程唯一 ID。
- `%&__TID%` 为当前线程 ID。
- 支持代理方式开线程，线程结束后代理退出。

Lua 重构提示：
- LuaJIT 本身不适合共享 Lua 状态多线程；建议用进程、job、消息队列或严格隔离的 worker。

## TIME

功能点：
- 创建定时器控件。
- 可在 `_SUB/_END` 内外使用。
- 支持自动回收。
- 支持周期毫秒。
- 周期大于 0 表示即刻工作，0 表示暂停。
- 触发时执行 WinCMD 命令。
- 环境变量 `%定时器名%` 表示工作状态，0 暂停，非 0 已启动。
- 可通过 `ENVI @定时器=周期[;|,次数]` 暂停/启动并设置次数。
- 可通过 `ENVI @控件.*del` 或 `ENVI @定时器=-del` 删除。

Lua 重构提示：
- 映射为 event loop timer。

## TIPS

功能点：
- 在屏幕指定位置显示提示框。
- 在托盘处显示气泡提示框。
- 省略所有参数时清除无效托盘图标。
- `TIPS -` 清除气泡提示框。
- `TIPS *` 清除托盘栏图标和气泡提示框。
- 后缀 `*` 表示窗口私有 TRAY 控件。
- 更新时托盘图标可填 `**` 表示图标保持不变。
- 支持等待模式。
- 支持程序结尾强制保持。
- 支持标题、正文、寿命、图标样式、屏幕位置、托盘图标、窗口 ID。
- 正文支持 `\n` 分行。
- `.` 表示不转换 `\n`。
- `#` 定界后正文可为任意字符。
- 图标样式 0 无图标、1 信息、2 警告、3 错误、4+ 托盘图标。
- 屏幕位置前导 `@`，`@A` 表示箭型提示框，否则方型提示框。
- 托盘图标可为 `[文件名][#数字]`，省略文件名用 PECMD 资源图标。
- 支持 `-dummy` 设置或清除 TIPS 模拟，`?` 只对 Win10 有效。

Lua 重构提示：
- 映射为 `ui.toast`、`tray.balloon`。

## UPNP

- 内嵌新版 BartPE.EXE 执行代码，无需外部 BartPE.EXE 文件。
- 前导 `$` 表示显示 BartPE.EXE 执行界面。
- 仅支持 NT5.x 系列 PE。
- 阻塞执行，完成后才继续下一条命令。

Lua 重构提示：
- 历史兼容功能，建议废弃或标记为 NT5 legacy。

## USER

- 参数为用户名和公司名。
- 逗号可用全角中文逗号或变量传入。

## WALL

功能点：

- 支持环境变量。

- 可在命令行和配置文件使用。

## WAIT

功能点：
- 等待指定毫秒数后继续。
- 等待期间不阻塞图形界面。
- 支持前导 `-`，在指定等待时间内遇任意按键即中止等待。
- 时间为 0 时暂停，任意键继续，可用于检测按键。
- 时间为 `-N` 可检测用户按键。
- 按键返回字母数字或 `0xNN`，默认变量为 `PressKey`。
- `&&PressKey.Hex` 返回 `0xNN` 按键或 -1。
- 支持 `-scanall` 返回全部 256 键扫描表。
- 支持 `-cont` 不遗漏两次调用之间的按键。
- 支持 `-pad` 区分小键盘数字。
- 支持释放内存。
- 支持等待进程结束。
- `*` 可表示父进程，`**` 表示祖父进程，带数字则以该进程为起点。
- 支持等待线程号 `=tid`。
- 支持等待句柄 `$handle`。
- 支持等待所有子线程。
- 等待后可依次删除指定文件或目录。
- 删除模式会定期重试直到删除成功。
- 等待后可执行一条 PECMD 命令。
- 支持系统代理模式和系统直接代理执行命令。
- 系统代理模式中文件名需要绝对路径。
- 支持等待进程、执行命令时不转移当前目录。
- 时间大于 0 小于 1 表示处理积压消息个数。
- 可用于设置 SHELL 监视日志。

Lua 重构提示：
- 映射为 `sleep`、`input.wait_key`、`process.wait`、`thread.wait`、`handle.wait`、`deferred_delete`、`system_agent`。

## WRIT

功能点：
- 以字符串/行方式修改文本文件。
- 支持编码：Unicode、Unicode BE、UTF-8、GBK、BIG5、ANSI、数字代码页。
- BOM 优先。
- 支持 `*fix`，孤立回车也当换行。
- 支持 `*-nl`，后面不追加换行。
- 支持 `*v` 写到变量。
- 支持 `*fv` 内容参数是变量名。
- 支持 `*c` 先清空。
- 文件名 `-` 表示标准输出。
- 文件名 `--` 表示标准错误输出。
- 修改位置支持 `$`、`+`、`-` 和行号。
- `$` 表示展开文件数据中的环境变量。
- `+` 表示在指定行插入。
- `-` 表示删除指定行。
- 无 `+/-` 表示替换指定行。
- 行号 0 表示最后一行。
- 不合理行号不操作。
- 支持特殊设备路径和 ImDisk 路径。
- 只适用于纯文本文件。
- `CONOUT$` 可用于调试终端。
- 不能套在 `FIND`、`IFEX`、`TEAM` 内，必须单独一行。

Lua 重构提示：
- 映射为 `text.write_line/insert/delete/replace`、`encoding.encode`、`stdout/stderr.write`。

## 并行符 `\`

功能点：
- 一行第一个非空字符为 `\` 时，并入上一行。
- 用于避免一行过长。
- `\` 及以前空字符、后面的一个空字符如果存在，均会丢弃。

Lua 重构提示：
- 属于兼容脚本预处理器能力。

## DLL 版 PECMD

功能点：
- 支持 `PECMD.DLL` DLL 版入口。
- 可通过 `CALL $**c **1 PECMD.DLL,main,命令行` 调用。
- 可通过 `CALL $**1 PECMD.DLL,main_,命令行` 调用。
- 可通过 `rundll32.exe PECMD.DLL,main 命令行` 调用。

Lua 重构提示：
- 如需 DLL 入口，应单独设计 C ABI 或 LuaJIT host DLL，不建议自然继承旧调用约定。
