## ENVI/SET 续：系统探测与底层状态

功能点：
- 支持直接修改或查询 PECMD 内存变量。
- 支持从注册表刷新环境变量。
- 支持查询父进程 PID。
- 支持查询当前进程是否管理员。
- 支持设置窗口或控件背景进度，包括百分比、样式、背景色、进度色、文本色、文本。
- 支持私秘模式，内存用完后清零。
- 支持关闭和恢复 x64 文件系统重定向。
- 支持查询 Windows 版本，且可按系统核心文件或指定文件产品/文件版本修正判断。
- 支持查询当前是否处于 PE 环境。
- 支持查询程序或系统位数，返回 x86/x64/ia64 和数字状态。
- 支持查询固件变量。
- 支持设置、删除或按不同编码/字节模式操作固件变量。

Lua 重构提示：
- 应拆为 `process.parent_pid`、`security.is_admin`、`wow64.disable_redirection`、`sys.version`、`sys.is_pe`、`firmware.variable`、`ui.progress`。
- “PECMD 内存变量”属于旧解释器内部能力，Lua 重构通常应废弃或替换为明确的 Lua 状态对象。

## EXEC

功能点：
- 执行 EXE、BAT、CMD 程序。
- 支持等待或非等待执行。
- 支持等待界面初始化。
- 支持把最近一级 PE 变量继承为环境变量。
- 支持隐藏、显示、最小化、最大化窗口。
- 支持设置工作目录，可取执行文件目录或自身目录。
- 支持通过 `-exe` 指定真实执行文件名，从而执行非标准后缀可执行文件，后续 EXE 路径仅作为命令行片段。
- 支持设置进程优先级：实时、高、较高、普通、较低、低/空闲。
- 支持安全降权执行。
- 支持 Job 对象方式执行，确保子孙进程随父进程终止。
- 支持指定账号和密码运行程序。
- 支持捕获标准输出。
- 支持捕获标准错误。
- 支持合并捕获标准输出和标准错误。
- 支持原始二进制捕获。
- 支持把输出保存到变量。
- 支持只取第一行输出，并在收到一行后终止进程。
- 支持去掉换行、合并成一行、去掉最后换行。
- 支持把输出实时追加或写入控件，例如 `EDIT`、`MEMO`、`LABE`。
- 支持管道模式，包含二进制命令管道和重定向：`>`、`>>`、`<`、`2>`、`>&`。
- 支持文档模式，通过 ShellExecute 打开非可执行文件。
- 文档模式支持 `open`、`edit`、`explore`、`find`、`print`、`properties`、`runas`。
- 支持无窗口执行。
- 支持禁止等待光标。
- 支持接管子进程标准输入输出。
- 支持以后台桌面/WinLogon 桌面执行。
- 支持修改目标进程关机函数，挂接 `ExitWindowsEx`，配合 `SHUT` 接管开始菜单关机。
- 支持写入注册表 Run 自动运行项。
- 支持克隆一个 PECMD 运行变量中的脚本。
- 支持指定另一个 PECMD.EXE 执行克隆脚本。
- 支持克隆脚本时按当前日志状态附带子日志。
- 支持 32 位 PECMD 不解除 x64 文件系统重定向限制。
- 支持克隆执行内部命令。
- 支持按 `SHEL` 方式执行并锁定 shell。
- 支持等待超时，超时时可杀进程或不杀进程，可杀子进程。
- 支持 System 权限执行、激活窗口、继承环境、跳过 UAC、保留/切换当前目录。
- 支持服务到桌面、用户服务到桌面、禁止切换桌面。
- 支持安装为服务和移除服务。
- 服务安装模式支持延时、等待/不等待、隐藏、杀窗口、空闲后执行、无 Job、无界面交互、切换显示、以桌面用户 ID 运行。
- 服务安装模式支持 `HERE_IS_SERVICE0` 内嵌服务参数，最多 100 字符。
- 支持执行内嵌程序。
- 支持从 EXEDATA 资源、CMPS 压缩资源、CAB 子文件、PE 变量、UD 文件、自解压包执行程序。
- 支持内存直接执行或释放到自动删除临时文件后执行。
- 支持更安全删除临时文件，并可用占位符替代目录或文件名。
- 支持非数字资源名和资源目录。
- 支持弹出文件右键菜单。
- 支持执行文件右键菜单指定条目。
- 支持保存 PID 或进程句柄到变量。
- 支持指定源编码。
- 支持 Unix 到 DOS 文本转换。

隐藏功能：
- 可不用 `EXEC` 直接运行外部程序，此时为等待模式。
- 没有命令词且后缀像脚本文件时按 `LOAD`，否则按 `EXEC`。
- 等待模式会把程序退出码写入 `&&ERRORLEVEL`、`ERROR`、`&&ERROR`。
- `SHELL` 也是通过此命令载入。
- 开关前导冲突时，最后出现的有效。
- 前导符和文件名之间可加空格分断。
- 文档模式如果要带参数，文件名必须用引号，否则全部会当作一个文件名。

Lua 重构提示：
- 现有 `win-utils.process`、`process.popen`、`win-kit.pecmd_logic` 已覆盖部分执行、等待、捕获、杀进程能力。
- `proc_utils` 仅作历史行为对照，新实现不依赖它；`exec_as_system`/System/UAC 执行在 WinPE 目标下不迁移。
- 仍需逐项确认：服务安装模式、内嵌程序执行、管道二进制、右键菜单、WinLogon 桌面、ExitWindowsEx hook。

## EXIT

功能点：
- 退出当前 `CALL` 子过程。
- 退出当前 `LOAD` 配置文件过程。
- 退出当前 `LOOP` 循环。
- 退出当前 `FORX` 循环。
- 跳到当前代码块尾部。
- 继续 `LOOP/FORX` 下一轮。
- 从窗口函数立即返回窗口控制。
- 设置命令错误码。
- 可从 PE 变量地址设置错误码。
- 可返回当前错误码。
- 错误码是线程独立的。

Lua 重构提示：
- Lua 可用 `return`、`break`、协程取消、错误码上下文替代。

- 支持最小值、最大值、剩余值，单位 MB。
- 支持单独指定某个参数强制设置。

Lua 重构提示：
- 现有 `win-utils.disk.fbwf` 是候选实现。

## FDIR

功能点：
- 返回文件所在目录，不含结尾反斜杠。
- 支持相对路径，按 PECMD 工作目录或配置文件目录解析。
- 支持返回绝对目录。
- 支持返回文件完整路径。
- 支持短文件名和长文件名。
- 支持如有空格则加引号。
- 支持返回 Volume 路径格式。
- 支持返回 `\?\Harddisk#Partition#` 分区格式。
- 支持 `\?\GLOBALROOT\Device\...`、`\?\HarddiskVolume...`、`\?\ImDisk...` 等设备路径。

Lua 重构提示：
- 映射为 `path.dirname`、`path.fullpath`、`path.short_name`、`disk.volume_path`。

## FDRV

功能点：
- 返回文件或目录所在分区盘符。
- 省略路径时返回系统所有盘符。
- 可返回带冒号、带空格分隔、仅字母等不同盘符列表。
- 可返回系统所有 DOS 设备。
- 可查询符号链接对象的所有名和最终名。
- 可查询卷标、文件系统、序列号、最大文件名长度、文件系统标志、UUID。
- 可设置卷标。
- 支持反序、排除 A/B 软盘、查询空闲盘符。

Lua 重构提示：
- 映射为 `disk.volume.list`、`disk.volume.info`、`disk.volume.set_label`、`sys.dos_device`。

## FEXT

功能点：

- 支持特殊设备路径。

## FIND

功能点：
- 根据条件表达式执行真/假两组命令。
- 支持嵌套条件。
- 支持组合条件，关系运算为与、或、异或。
- 支持条件取反。
- 支持内存总数判断，单位 MB 或字节。
- 支持磁盘总空间判断，单位 MB 或字节。
- 支持按键状态判断。
- 支持进程存在判断，进程名可含通配符。
- 支持字符串比较，区分或不区分大小写。
- 支持 PE 变量名直接比较。
- 支持数值比较，浮点或整数。
- 支持查询内存、磁盘、按键条件值。
- 支持查询进程列表，返回 PID、PPID、内存、CPU 时间、总时间、文件名、命令行。
- 支持查询系统 CPU 空闲时间、总时间、CPU 个数、时钟信息。
- 支持查询窗口列表，返回序号、窗口 ID、控件 ID、父窗口 ID、线程 ID、进程 ID、类型、标题。
- 支持按父窗口、窗口标题、控件 ID、进程 ID、线程 ID、可见/不可见、窗口类名过滤窗口。
- 支持查询菜单和子菜单信息。
- 支持无 ELSE 简写。
- 支持特殊设备路径容量判断。

Lua 重构提示：
- 现代 Lua 可用 `if`，但条件探测能力应拆成 `sys.memory`、`disk.space`、`input.key_state`、`process.find`、`window.find`、`menu.query`。

## FORX

- 支持递归子目录和子目录级数。
- 支持按名称排序和反序。
- 支持按大小范围和对齐过滤。
- 支持排除 A/B 移动盘。
- 支持当前盘符优先。
- 支持搜索所有分区或逆序搜索所有分区。
- 支持仅搜索文件、仅搜索目录、搜索目录和文件。
- 支持执行次数限制。
- 支持变量代表当前匹配路径。
- 支持遍历变量表。
- 支持遍历多行 PE 变量。

- 支持命令为空时返回找到的数量或执行次数。
- 支持命令为 `*` 时返回文件列表。
- 支持隐藏属性文件目录。

## HASH

功能点：
- 计算文件 CRC32、MD5、SHA1、SHA256。

- 省略变量时显示并复制到剪贴板。
